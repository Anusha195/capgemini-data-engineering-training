# DAY 3 – SQL Concepts (CASE WHEN & Window Functions)

This document summarizes the concepts learned on Day 3 of my data engineering training.

---

## Topics Covered

* CASE WHEN
* Nested CASE WHEN
* Window Functions

  * ROW_NUMBER()
  * RANK()
  * DENSE_RANK()
  * PARTITION BY
* NULL Handling (basic understanding)
* REGEX (introduction)

---

## What I Learned

* CASE WHEN is used to apply conditional logic in SQL.
* Nested CASE WHEN helps handle multi-level conditions.
* Window functions are used for ranking and analysis without reducing rows.
* Gained basic understanding of handling NULL values.
* Explored introductory concepts of REGEX for pattern matching.

---

## Key Takeaways

* CASE WHEN is useful for data transformation and categorization.
* Window functions are important for analytical queries.
* NULL handling and REGEX require further practice and deeper understanding.

## Conclusion

Day 3 focused on applying SQL logic and understanding analytical functions, along with exposure to additional concepts like NULL handling and REGEX.
